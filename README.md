## 🧠 About This Portfolio

I build and ship **end-to-end computer vision systems** — from raw dataset ingestion to deployed inference demos. Projects span medical imaging, agricultural AI, real-world image restoration, and scene understanding, all implemented with production-conscious practices: mixed precision training, Test Time Augmentation, Grad-CAM explainability, and Gradio-based deployments.

---

## 🔄 Computer Vision Workflow

| Stage | Details |
|---|---|
| **Data Preparation** | Dataset loading, augmentation, normalization, and train/val/test splitting |
| **Model Selection** | Choosing architectures (CNN, EfficientNet, MobileNet, ViT) based on task |
| **Training** | Mixed precision training, label smoothing, learning rate scheduling |
| **Optimization** | Test Time Augmentation (TTA), hyperparameter tuning, gradient loss |
| **Evaluation** | Accuracy, mAP, PSNR, confusion matrix, Grad-CAM visualizations |
| **Deployment** | Gradio demos, HuggingFace Spaces, model export |

## 🚀 Projects

| Project | Description | Skills |
|---|---|---|
| [**Plant Disease Classification**](https://github.com/pankajmohansahu/plant-pathology-38class-efficientnetv2_computer_vision) | 38-class plant disease image classification using EfficientNetV2. Achieved **99.88% validation accuracy** using transfer learning, TTA, mixed precision training, and label smoothing. IIT Madras DLP Competition. | EfficientNetV2, PyTorch, TTA, Mixed Precision, Label Smoothing |
| [**Colorectal Histology Classifier**](https://github.com/pankajmohansahu/colorectal-histology-classifier_computer_vision) | 9-class colorectal tissue classification fine-tuned on the NCT-CRC-HE-100K medical imaging dataset using EfficientNet-B0. Achieved **99.84% validation accuracy**. Includes an interactive Gradio demo for inference. | EfficientNet-B0, PyTorch, Medical Imaging, Gradio, Transfer Learning |
| [**Low-Light Image Restoration**](https://github.com/pankajmohansahu/low-light-image-restoration_Computer_Vision) | Real-world low-light image restoration using MIRNet architecture. Achieved **27 dB PSNR** on real low-light images. Implements multi-scale residual blocks with PixelShuffle upsampling and TTA for enhanced restoration quality. | MIRNet, PyTorch, PSNR, PixelShuffle, TTA |
| [**Agricultural Pest Detection**](https://github.com/pankajmohansahu/dlp-week10-agricultural-pest-detection_Computer_Vision) | YOLOv8-based object detection model for 23-class agricultural pest classification. Achieved **mAP@0.5 of 0.6746** and secured **Rank 6** on the public leaderboard. IIT Madras DLP Week 10 Competition. | YOLOv8, Object Detection, mAP, PyTorch, IIT Madras DLP |
| [**Stereo Depth Estimation**](https://github.com/pankajmohansahu/dlp-week11-depth-estimation_Computer_Vision) | Depth estimation from scratch using stereo RGB images. Trained with gradient loss and mixed precision. Achieved a leaderboard score of **0.74281** on the IIT Madras DLP Week 11 competition. | PyTorch, Depth Estimation, Gradient Loss, Mixed Precision, Stereo Vision |
| [**Scene Image Classifier**](https://github.com/pankajmohansahu/scene-image-classifier-cnn_Computer_Vision) | Scene image classification using MobileNetV2 transfer learning. Built as part of IIT Madras DLSA Lab covering fine-tuning, feature extraction, and multi-class classification on scene datasets. | MobileNetV2, TensorFlow, Transfer Learning, CNN |
